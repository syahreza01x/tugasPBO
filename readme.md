## Kontrol Pemain

### **Player 1**
- **Gerak:**  
  - **W** = Atas  
  - **A** = Kiri  
  - **S** = Bawah  
  - **D** = Kanan
- **Tombol Skill:**  
  - **E** (default, bisa diubah di Settings)
- **Skill:**  
  - **Time Stop**  
    Menghentikan pergerakan peluru untuk sementara waktu.

---

### **Player 2**
- **Gerak:**  
  - **Arrow Up** = Atas  
  - **Arrow Left** = Kiri  
  - **Arrow Down** = Bawah  
  - **Arrow Right** = Kanan
- **Tombol Skill:**  
  - **END** (default, bisa diubah di Settings)
- **Skill:**  
  - **Area Clear**  
    Menghapus peluru di sekitar karakter.
